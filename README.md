Ejecución del proyecto 

Prerrequisitos:
tener instalado los siguientes programas
-java JDK
-java JRE
-Apache Netbeans

Abrir el Netbeans y el el menu <<file>>, elegir <<Open project>>

Una vez abierto el proyecto se tiene que ejecutar con el boton <<Run project>>
